This endpoint will tag sentences' items with POS tags.

{{ 'POST /tagger' }}

Expected input

The request payload must be a valid JSON.
[ "I sentence to tage.", "Another sentence to tag!" ]
The tagset used in this endpoint is from Penn Treebank Project.
You can append symbols to the query string. The API will return POS tags that are used in Penn treebank tagset instead of the descriptive strings.

Example - tagging with Penn symbols

Request

POST localhost:5000/lemma?symbols

Response

Time is expressed in seconds
{ "input": [ "I yelled at him, but he didn't listen..." ], "results": [ [ { "item": "I", "partOfSpeech": "PRP" }, { "item": "yelled", "partOfSpeech": "VBD" }, { "item": "at", "partOfSpeech": "IN" }, { "item": "him", "partOfSpeech": "PRP" }, { "item": ",", "partOfSpeech": "," }, { "item": "but", "partOfSpeech": "CC" }, { "item": "he", "partOfSpeech": "PRP" }, { "item": "did", "partOfSpeech": "VBD" }, { "item": "n't", "partOfSpeech": "RB" }, { "item": "listen", "partOfSpeech": "VB" }, { "item": "...", "partOfSpeech": ":" } ] ], "time": 0.0009036064147949219 }
